# GateCtr Docs

**One gateway. Every LLM.**

GateCtr sits between your app and any LLM provider. One endpoint swap. No code changes. Full control over costs, routing, and observability.

## What you get

- **-40% tokens** — Context Optimizer compresses prompts automatically
- **Budget Firewall** — Hard caps per project. No surprise invoices.
- **Model Router** — GateCtr picks the right LLM for each request
- **Analytics** — Every token, every cost, real-time
- **Webhooks** — Push events to Slack, Teams, or any URL
- **RBAC** — Team access control with roles
